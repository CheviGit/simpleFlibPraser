Основная задача преобразовать данные из локальной базы книг Флибусты в целевую модель классов для дальнейшего использования в основном приложении.
Из сторонних бибилиотек в реализации используется jsoup.org, для парсинга и  slf4j.org для логирования. Из Java core использованы I/O, Коллекции, Регулярные выражения.
 Spring IOC не использовался в виду слабой связности между объектами.
Возможности Hibernate так же избыточны для маппинга 2-3 фиксированных таблиц.
